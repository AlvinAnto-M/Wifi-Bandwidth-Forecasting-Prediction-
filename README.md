# 📶 WiFi Bandwidth Forecasting using Machine Learning

## 📌 Overview

WiFi Bandwidth Forecasting is a Machine Learning project that predicts future network bandwidth usage based on historical WiFi traffic data. Accurate bandwidth prediction helps network administrators optimize resource allocation, improve network performance, minimize congestion, and ensure a better user experience.

This project was developed as part of an internship to demonstrate the application of machine learning techniques in network traffic analysis and forecasting.

---

## 🚀 Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Historical bandwidth usage analysis
- Machine Learning-based bandwidth prediction
- Model performance evaluation
- Visualization of actual vs predicted bandwidth usage

---

## 🛠️ Tech Stack

## 📊 Dataset

The project uses historical WiFi bandwidth consumption data collected from a campus network.

The dataset includes network usage information that enables the model to learn bandwidth consumption patterns and forecast future bandwidth requirements.

---

## ⚙️ Workflow

1. Import the dataset
2. Clean and preprocess the data
3. Perform Exploratory Data Analysis (EDA)
4. Select relevant features
5. Train the Machine Learning model
6. Evaluate model performance
7. Predict future bandwidth usage
8. Visualize prediction results

---

## 📈 Results

The developed model successfully learns historical bandwidth usage patterns and generates future bandwidth forecasts. These predictions can assist in:

- Network capacity planning
- Efficient bandwidth allocation
- Congestion reduction
- Performance optimization
- Better network management
